  • Cheat sheet - HTML

    1. Elements and Structure

      1. Structure: html>body/footer>title, h1-6; div, span, p, br;
      2. Lists: ol/ul, li,
      3. Media: video, img, embed, a
      4. Format: br, em, strong

    2. Tables

      1. Structure: table>thead/tbody/tfoot>tr>td
      2. Attributes: row/colspan

    3. Forms

      1. Form: method, action
      2. Input: text/sunmit; checkbox/radio/range (min,max,step); name; value
      3. Select (name) > option (value)
      4. Datalist
      5. Label: for"=id of input"
      6. Textarea: rows, cols, name

    4. Semantic

      1. header>nav>main>body>footer
      2. Media: video, audio, embed; figure, figcaption
      3. Section, article, aside

  • Cheat sheet - CSS

    1. Selectors: specificity - !important/id/class/type/
    2. Hierarchy: descendants, groups, chaining.
    3. Basics: link to css file (href,rel,type); declarations, rule sets
    4. Formatting: color name keywords, font family, text align, font weight, background color ....
